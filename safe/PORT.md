The workspace keeps the original libxml2 C tree under `original/` and the Rust-backed port under `safe/`. The Cargo package is an implicit one-member workspace because `safe/Cargo.toml` has no `[workspace]` table. `safe/Cargo.toml:1-22` defines package `libxml2`, version `2.9.14`, Rust edition `2021`, minimum Rust `1.82`, no Cargo features, library target `xml2` with crate types `staticlib` and `rlib`, and binaries `xmllint` and `xmlcatalog`. No generated Rust binding pipeline is wired in.
